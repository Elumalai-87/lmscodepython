Geometry Questions

1. What is the area of a circle with a radius of 7cm?

Answer: Area = πr² = 3.1416 * (7)² = 153.94 cm²

2. Find the perimeter of a rectangle with length 10cm and width 5cm.

Answer: Perimeter = 2 * (length + width) = 2 * (10 + 5) = 30cm
