Student Projects

Here are some fun and educational project ideas you can try:

  • 1. Personal Portfolio Website

    Create a simple website to showcase your resume, projects, and contact details using HTML & CSS.

  • 2. To-Do List App

    Use JavaScript to create an interactive task list with add/delete functionality.

  • 3. Calculator

    Build a basic calculator using JavaScript that performs arithmetic operations.

  • 4. Quiz Game

    Create a simple multiple-choice quiz using JavaScript and track the score.

  • 5. Weather App

    Fetch weather data from an API and display it based on user input (advanced level).
