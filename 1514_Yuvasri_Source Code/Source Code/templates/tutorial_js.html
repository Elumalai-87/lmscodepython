Welcome to the JavaScript tutorial. Learn how to make your web pages interactive and dynamic!



Introduction

JavaScript is the programming language of the web. Learn how it can make websites interactive.

Q1: What is JavaScript used for?

A: JavaScript is used to create interactive and dynamic web pages. It can be used for event handling, DOM manipulation, and more.

JS Syntax

Understand how to write valid JavaScript statements, blocks, and scripts.

Q1: What is the correct syntax to declare a variable in JavaScript?

A: The correct syntax is `let variableName;`, `const variableName;` or `var variableName;`.

Variables

Learn how to declare variables using var, let, and const.

Q1: What is the difference between let and const?

A: The let keyword allows variable reassignment, while const declares a constant value that cannot be reassigned.

Data Types

JavaScript supports strings, numbers, booleans, objects, undefined, and more.

Q1: What are the primitive data types in JavaScript?

A: The primitive data types in JavaScript are: String, Number, Boolean, Null, Undefined, Symbol, and BigInt.

Operators

Explore arithmetic, assignment, comparison, and logical operators in JavaScript.

Q1: What is the difference between == and === in JavaScript?

A: The == operator checks for equality of values, whereas the === operator checks for both equality of value and type.

Functions

Learn to create and invoke functions to structure reusable code blocks.

Q1: How do you define a function in JavaScript?

A: You define a function using the function keyword. Example: function myFunction() { ... }.

Events

Handle user interactions like clicks, form submissions, and keyboard input using event listeners.

Q1: How do you add an event listener to a button in JavaScript?

A: You can add an event listener using addEventListener. Example: button.addEventListener('click', function() { ... });.

DOM Manipulation

Access and change HTML elements dynamically using JavaScript and the Document Object Model.

Q1: How do you change the text content of an HTML element in JavaScript?

A: You can change the text content using element.textContent = 'New text';.

Loops

Use for, while, and do...while loops to repeat code.

Q1: How does a for loop work in JavaScript?

A: A for loop allows you to repeat a block of code a set number of times. Example: for (let i = 0; i < 5; i++) { ... }.

Arrays

Store multiple values in a single variable using arrays and learn how to manipulate them.

Q1: How do you create an array in JavaScript?

A: You can create an array using square brackets. Example: let arr = [1, 2, 3];.
